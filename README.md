# Blockchain-Based Academic Credential Verification Platform

A decentralized academic credential verification system that allows universities to issue certificates, students to manage credentials, and employers to verify authenticity using blockchain technology.

## Technology Stack

- React
- Vite
- TailwindCSS
- FastAPI
- PostgreSQL
- Solidity
- Hardhat
- MetaMask
